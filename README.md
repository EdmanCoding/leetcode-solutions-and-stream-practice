# Java Interview Preparation

This repository contains my Java interview preparation materials, including LeetCode solutions, algorithm notes, test cases, and Java Streams practice exercises.

## Goals

- Improve problem-solving skills
- Build a strong understanding of common interview patterns
- Practice writing clean and testable Java code
- Prepare for technical interviews and live-coding sessions

## Tech Stack

- Java 25
- Maven
- JUnit 5
- AssertJ

## Repository Structure

### LeetCode Problems

Solutions are organized by groups of problems.

Examples:

- Two Sum
- Valid Parentheses
- Contains Duplicate
- Best Time to Buy and Sell Stock
- Binary Search
- Merge Sorted Array
- Move Zeroes
- Linked List Cycle
- Invert Binary Tree
- Maximum Depth of Binary Tree
- Balanced Binary Tree
- LRU Cache
- and others

For many problems, multiple approaches are included:

- initial solution
- improved solution
- optimized solution
- alternative approaches from discussions

### Tests

Each problem contains dedicated unit tests covering:

- base cases
- edge cases
- boundary conditions
- invalid input scenarios
- large input scenarios

The goal is not only to solve problems but also to verify correctness systematically.

### Notes and Patterns

The repository contains notes about:

- algorithmic patterns
- loop invariants
- dynamic programming concepts
- complexity analysis
- problem-solving strategies

### Java Streams Practice

A separate package contains stream exercises focused on:

- `map`
- `filter`
- `flatMap`
- `groupingBy`
- collectors
- downstream collectors
- comparators
- reductions
- real interview-style stream tasks

These exercises are used to prepare for Java live-coding interviews.

## Why This Repository Exists
This repository serves as my personal knowledge base and interview preparation workspace.

The focus is not only on solving problems, but also on understanding:

- why a solution works

- the underlying invariant

- time complexity

- space complexity

- alternative approaches

- common interview discussion points
